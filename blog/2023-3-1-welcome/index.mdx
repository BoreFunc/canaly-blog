## What is Canaly?
Canaly is a project for analyzing CAJ files with open source tools. Our goal is to provide a set of open source tools for analyzing CAJ files, and to provide basic infomation for the community to develop tools for CAJ files.

## What is CAJ?
CAJ is a file format used by Chinese Academy of Sciences (CAS) for storing scientific articles. It is a proprietary format, to read CAJ files, you need to install either proprietary software or commercial software.
